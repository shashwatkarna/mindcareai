# MindCare AI

A secure, privacy-focused platform for mental health support and wellness tracking.

## Features
- **Privacy First**: Secure user authentication and data handling.
- **Wellness Tracking**: Track your daily mood and activities.
- **Resources**: Access a library of mental health resources (to be implemented).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
